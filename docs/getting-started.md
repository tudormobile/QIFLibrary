# Getting Started
## Install the package
```
dotnet add package Tudormobile.QIFLibrary
```
### Prerequisites
**NONE**

### Dependencies
- None

## Key Features
- Parse QIF files into document model
- Parse QIF data streams (and strings) into QIF documents and records
- Asynchronous parsing for large data sets.

### Feedback
